This project is a modern and responsive eCommerce frontend application designed to deliver a seamless online shopping experience. It enables users to browse products, view detailed information, manage their shopping cart, and proceed through a smooth checkout flow with an intuitive user interface.

Features

Product listing and browsing

Product detail pages

Dynamic cart management

Responsive design for all devices

API-based data rendering
